# Backend for AI Cost Intelligence Copilot

This folder contains the FastAPI backend, agent modules, RAG pipeline, graph builder, and financial impact engine.

## Structure
- agents/: Spend, SLA, Resource, Finance agents
- rag/: Retrieval-Augmented Generation pipeline
- graph_builder/: Graph construction logic
- api/: FastAPI endpoints
- financial_impact.py: Cost impact calculations
- main.py: FastAPI app entrypoint
